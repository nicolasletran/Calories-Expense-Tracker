# Calorie & Protein Tracker Web App

A modern web application for tracking daily calorie and protein intake with beautiful visualizations.

## Features
- Add, edit, and delete meal entries
- Track calories and protein
- Set and monitor protein goals
- Weekly summary with charts
- Mobile-responsive design
- Real-time updates

## Installation

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
